data-linked visualisations

Fluid is a new programming language which utilises program analysis/data provenance techniques to connect visualisations to data sources in a fine-grained way. For example, the Fluid runtime will link a visual selection in a bar chart to the smallest “data selection” containing the data needed to compute that bar. (A data selection is a restricted view of a dataset where only relevant rows are shown, and relevant cells within that are highlighted.) As well as offering the first fully automated (and thus potentially pervasive) solution to this well-recognised problem in data visualisation, our approach to data linking is “transparent”, in that a view of the specific data elements which “explain” why the two visual elements are linked can always be made available.
